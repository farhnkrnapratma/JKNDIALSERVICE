# Assets Folder

Folder ini berisi assets untuk aplikasi mobile:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1242x2436)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Generate Assets

Anda bisa generate assets menggunakan:

1. **Figma/Canva** - Design custom
2. **Online tools** - https://appicon.co
3. **Expo asset generator**

## Quick Placeholders

Untuk testing, Expo akan generate placeholder otomatis jika file tidak ada.

Atau download dari:
- https://via.placeholder.com/1024x1024.png (icon)
- https://via.placeholder.com/1242x2436.png (splash)

## Recommended Sizes

- Icon: 1024x1024 PNG (transparent background)
- Splash: 1242x2436 PNG
- Adaptive Icon: 1024x1024 PNG (foreground layer)
- Favicon: 48x48 PNG
